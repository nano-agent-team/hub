# nano-agent-team hub

Public catalog of teams and agents for nano-agent-team.

## Structure

```
teams/
  {team-id}/
    team.json     — team manifest (id, name, description, agents)
    setup.json    — required config fields shown in setup wizard
agents/
  {agent-id}/
    manifest.json — agent manifest
    setup.json    — optional required config fields
```
